对象类型和原始类型的不同之处？函数参数是对象会发生什么问题？ ​

对象类型 ​

  • 对象类型存储的是(地址)指针:声明一个对象的时候计算机会在内存中为我们开辟一块空间来存放值
  • 赋值的时候,是赋值的地址:进行数据修改的时候,会改变原来的值

函数传参 ​

  • 函数传参是传递对象指针的副本
  • 在函数内部改变对象的属性会影响原来的值
