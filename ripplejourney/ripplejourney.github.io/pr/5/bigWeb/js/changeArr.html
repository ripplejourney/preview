数组哪些方法会改变自身 ​

ES5 ​

  • splice：删除数组元素或在目标位置插入新元素
  • sort：排序数组
  • reverse：逆序数组
  • push：在数组末尾添加元素
  • pop：删除数组末尾的元素
  • shift：删除数组开头的元素
  • unshift：在数组开头添加元素

ES6 ​

  • copyWithin：在当前数组内部，将指定位置的成员复制到其他位置
  • fill：使用给定值，填充一个数组
